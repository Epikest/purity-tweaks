# Purity Tweaks
## Installation
1. [Download ZIP](https://github.com/Epikest/purity-tweaks/archive/refs/heads/main.zip)
2. Put ZIP in .minecraft/resourcepacks
3. (optional) Rename to `§bPurity §dTweaks §8§ov1.0.0.zip`
4. Select in game:
  * Game Menu
    * Options...
      * Resource Packs...
        * Select the Purity Tweaks pack
